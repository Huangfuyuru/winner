### 1.使用变量  
sass使用$符号来表示变量
#### 1-1.变量声明  
`highlight-color:#F90`  
`$basic-border:1px solid black`  
当这个变量声明在{}内，我们不能在{}外使用这个命令，可以避免变量污染  
```
$nav-color:#F90;
nav{
    $width:100px;
    width:$width;
    color:$nav-color;
}
//编译后
nav{
    width:100px;
    color:#F90
}
```
